CSS中，all属性是一个简写属性，它可以重置选中元素的所有属性（除了unicode-bidi和direction）为初始值、继承值或者其他样式表来源的值¹²⁴。这个属性可以用来一次性控制多个CSS属性的值，但是它没有长格式版本或子属性¹³。例如，如果你想要让一个div元素的所有属性恢复到浏览器的默认设置，你可以这样写：


div {
  all: initial;
}

这样，div元素的所有属性（除了unicode-bidi和direction）就会变成初始值。如果你想要让一个div元素的所有属性继承自它的父元素，你可以这样写：

div {
  all: inherit;
}

这样，div元素的所有属性（除了unicode-bidi和direction）就会变成继承值。

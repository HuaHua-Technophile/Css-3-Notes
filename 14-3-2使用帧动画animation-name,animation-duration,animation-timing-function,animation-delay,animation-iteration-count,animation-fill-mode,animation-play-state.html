1. animation-name：指定要绑定到选择器的关键帧的名称。
  2. animation-duration：用于指定动画完成一个周期所花费的时间，可以用秒（s）或毫秒（ms）来表示。这个值必须是正数或者零，且单位是必须的。如果没有指定值，那么默认值是0s，此时动画仍然会执行（animationstart和animationend事件会被触发），但是否可见取决于animation-fill-mode属性的值
    3. animation-timing-function属性用于指定动画的速度曲线，即动画在每个周期内的变化速率。这个属性可以让动画更加平滑和自然
    1. linear：表示动画从头到尾保持匀速。
    2. ease：表示动画开始和结束时慢，中间时快，是默认值。
    3. ease-in：表示动画开始时慢，后面逐渐加速。
    4. ease-out：表示动画结束时慢，前面逐渐减速。
    5. ease-in-out：表示动画开始和结束时慢，中间时快。
    6. step-start：相当于steps(1,start)，表示动画在每个周期的开始时跳跃到结束状态。
    7. step-end：相当于steps(1,end)，表示动画在每个周期的结束时跳跃到结束状态。
    8. steps(int,start|end)：表示动画在每个周期内分成int个等长的步骤，每个步骤保持相同的状态，直到下一个步骤开始。第二个参数可以是start或end，表示步骤发生在每个区间的开始还是结束。
    9. cubic-bezier(n,n,n,n)：表示自定义的三次贝塞尔曲线，用四个参数来控制曲线的形状。参数必须在0到1之间。
  4. animation-delay：指定动画开始之前的延迟时间，可以用秒（s）或毫秒（ms）来表示。单位是必须的。如果没有指定值，那么默认值是0s，此时动画会立即开始。如果指定了负值，那么动画会从中途开始，相当于已经播放了一段时间
  5. animation-iteration-count:指定动画播放的次数，可以是一个正数或者无限次（infinite）。如果没有指定值，那么默认值是1，表示动画只播放一次。如果指定了非整数的值，那么动画会播放部分周期。例如，0.5表示动画只播放一半。负数是无效的
    6. animation-direction：于指定动画是否在每个周期内反向播放，或者交替正向和反向播放。如果没有指定值，那么默认值是normal，表示动画每次都正向播放
    1. normal：表示动画每次都正向播放，从头到尾。
    2. reverse：表示动画每次都反向播放，从尾到头。
    3. alternate：表示动画交替正向和反向播放，第一次是正向，第二次是反向，以此类推。
    4. alternate-reverse：表示动画交替反向和正向播放，第一次是反向，第二次是正向，以此类推。
    7. animation-fill-mode：指定动画执行之前和之后如何为目标元素应用样式。CSS动画默认不会影响元素在第一个关键帧之前或最后一个关键帧之后的样式。animation-fill-mode属性可以覆盖这种行为
    1. none：表示动画不会在执行之前或之后应用任何样式，元素会使用其他CSS规则来显示。这是默认值。
    2. forwards：表示元素会保留动画最后一个关键帧设置的计算值。最后一个关键帧取决于animation-direction和animation-iteration-count的值。
    3. backwards：表示元素会在动画开始之前（包括animation-delay期间）应用第一个相关关键帧设置的值。第一个相关关键帧取决于animation-direction的值。
    4. both：表示动画会遵循forwards和backwards的规则，从而在两个方向上扩展动画属性。
  8. animation-play-state：指定动画是否正在运行或暂停。你可以使用这个属性在:hover中,或JavaScript中暂停或恢复一个动画。恢复一个暂停的动画会从暂停的时间点开始，而不是从头开始.running(默认值)：表示动画正在运行。paused：表示动画已经暂停

animation属性是一个简写属性，可以同时设置上述所有的子属性，或者只设置其中的一部分。语法如下
animation: name duration timing-function delay iteration-count direction fill-mode play-state;

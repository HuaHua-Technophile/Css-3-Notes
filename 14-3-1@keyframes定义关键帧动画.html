@keyframes规则用于定义CSS动画的中间步骤，通过指定不同的关键帧（或者说路标）来控制动画序列中的样式变化。这比过渡（transition）能提供更多的控制能力¹²。
@keyframes规则的语法如下¹²： 

@keyframes animationname {
  keyframes-selector { css-styles; }
  keyframes-selector { css-styles; }
  keyframes-selector { css-styles; }
}

其中，animationname是动画的名称，用于animation-name属性来匹配动画和关键帧声明。keyframes-selector是动画持续时间的百分比，用于指定关键帧发生的时间点，可以是0-100%之间的数字，或者关键字from（相当于0%）和to（相当于100%）。css-styles是一个或多个合法的CSS样式属性。

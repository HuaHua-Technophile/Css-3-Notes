• background-position 可以使用关键字、百分比、长度或 calc() 表达式来定义坐标
  • background-position 的默认值是 0% 0%，表示背景图像的左上角与容器的左上角对齐
  • background-position 可以指定一个或多个值，如果指定多个值，表示有多个背景层，用逗号分隔

  • background-origin 是 CSS 属性，用于规定背景图片的原点位置的背景相对区域
  • background-origin 可以使用三个值：border-box、padding-box 或 content-box
  • background-origin 的默认值是 padding-box，表示背景图片的起始位置相对于 padding 区域
  • background-origin 可以和 background-position、background-size、background-clip 等属性配合使用，实现不同的背景效果

    • background-origin: border-box; /*默认值*/
    background-origin: padding-box;
    background-origin: content-box;

  • background-clip 是 CSS 属性，用于设置元素的背景（背景图片或颜色）是否延伸到边框下面
  • background-clip 可以使用四个值：border-box、padding-box、content-box 或 text
  • background-clip 的默认值是 border-box，表示背景从边框区域向外裁剪，超出部分将被裁剪掉。
  • background-clip 可以和 background-origin、background-position、background-size 等属性配合使用，实现不同的背景效果

    • background-clip: border-box; /*默认值*/
    background-clip: padding-box;
    background-clip: content-box;
    background-clip: text; -webkit-background-clip: text; color: transparent;
